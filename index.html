CheckSamples

Drug sample management solution that connects clinics and pharma sales reps

About

CheckSamples is a healthcare startup seeking to reduce pharmaceutical sample waste, improve administrative efficiency, ensure regulatory compliance, and connect clinics and pharma reps. As a founding member of the startup, I worked remotely with two co-founders to design and develop the system, first by building a protytype as an iPad app, and later as a fully featured web application.

Technology

The initial iPad app prototype was built using Swift and Objective-C. The web app is built using Meteor and Bootstrap and hosted on Galaxy. The database is hosted on MongoDB Atlas with full at-rest encryption for HIPAA compliance. Payments are processed via Stripe. Remote communication and management done primarily via Slack.

Nitroxbuddy

Smartphone gas analyzer app

About

Nitroxbuddy is a universal iOS app with an accompanying Apple Watch app extension that enables divers to monitor and analyze the gas content of their scuba tanks via a bluetooth sensor. I developed the iOS/watchOS app, including building a library for communicating with the wireless bluetooth sensor

Technology

Built the native app using Swift and Objective-C, managed the project using Jira.

USC Dive Log

Web based dive logging system for the University of Southern California

About

Online dive logging system enabling USC's dive safety officers and student/research divers to track dives and generate reports. I gathered requirements with the client, including a number of onsite visits to USC's research facility on Catalina Island where I observed the work flow. I built the custom dive logging system, improving on their prior manual system. I later ported, with some further customizations, for the University of Maine's scientific diving program.

Technology

Built the web system using PHP, Bootstrap and MySQL, and hosted on USC private servers. Used Trello for project management.

Photo Plot Layers

Data recording app for the University of California, Santa Cruz

About

Custom app developed for UCSC which allows recording of data collected by research divers. Gathered requirements from client and implemented the app to adhere to their internal "layered photo plot" data recording protocol. The app also relies on a server back end for data upload.

Technology

Built the native app using Swift and Objective-C. Built the server back end using Node and MySQL, and hosted on UCSC private servers. Managed the project using Jira, and maintained ongoing communication with remote client via regular video conferencing.
